## コマンドと説明 使用方法
| コマンド | 説明 | 使い方 |
| ------- | ----------- | ----- |
| `skill` | スキルを確認できるメニューを表示します | `/skill` |
| `money` | 経済要素の残高を表示します | `/money` |
| `sit` | 座れます (シンプル) | `/sit` |
| `qs create` | チェストショップを作成します | 売りたいアイテムを持って、視線をショップにしたいチェストに向けて `/qs create [価格]` |
| `qs buy` | ショップを買取モードにします | 視線をショップチェストに向けて `/qs buy` |
| `qs sell` | ショップを販売モードにします | 視線をショップチェストに向けて `/qs sell` |
| `witp` | 無限アスレを生成します | `/witp` |
